# Abrir-o-Chrome-como-Usu-rio
Abrir o Chrome como Usuário
Esse fluxo permite abrir o navegador Chrome como se fosse o usuário local da máquina.

Sobre este fluxo de automação:
Para tal é necessário apontar o driver do Chrome que deverá ser usado.
Isso deverá ser feito tanto no Studio quanto no Agente, mas dessa maneira você poderá usar complementos do Chrome.
O Navegador será aberto de forma Anônima.
